#### 2.MD5

> MD5 消息摘要算法,可以产生一个128位的散列值, 用于确保信息传输的一致性;
>
> 可以被加以破解, 一般用于验证文件的完整性;

- python中实现MD5加密:

  - ```python
    import hashlib
    
    hl = hashlib.md5()
    hl.update('test'.encode(encoding='utf-8')) # 根据字符串生产摘要
    hl.hexdigest()  # 以十六进制字符串返回当前摘要
    ```
